Events Queue
Events Queue : The events queue collects the events coming in from external sources (the user, interrupts etc.), or the various requests from the processes that need to be handled (such as requests for resource allocation).
CPU
The CPU : The processes currently being executed appear in the CPU.
I/O Pool
The I/O Pool : The processes which are currently using I/O resources appear in the I/O Pool.
Ready Pool
The Ready Pool : The ready pool is where the processes wait before they are allowed to be executed.
Terminated Processes
Terminated Processes : The processes which have been terminated appear here.
